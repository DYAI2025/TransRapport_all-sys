# TransRapport Terminologie

Test terminology document for validation.

## ATO_ (Atomic Marker)

**ATO** 路 Atomic markers represent the smallest units of analysis in the TransRapport pipeline.

## SEM (Semantic Marker)

**SEM** 路 Semantic markers group atomic markers into meaningful semantic units.

## CLU (Cluster Marker)  

**CLU** 路 Cluster markers aggregate semantic units into larger conceptual clusters.

## MEMA (Memory Marker)

**MEMA** 路 Memory markers represent long-term patterns and recurring themes.

## CLI Commands

### me docs validate

Validate documentation files for consistency and completeness.

### me docs status

Get status information about documentation files.
